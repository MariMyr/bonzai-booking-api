# Bonz.ai Booking API. Gruppexamination.

Detta är ett serverless API för Bonz.ai:s hotellbokningar. Projektet använder AWS Lambda, API Gateway, DynamoDB, Serverless.

## Endpoints
| Method  | Endpoint         | Beskrivning                          
|---------|------------------|------------------------------|
|  POST   |  /bookings/      |  Göra en bokning             |
|  PUT    |  /bookings/{id}  |  Ändra på en bokning         |
|  DELETE |  /bookings/{id}  |  Avboka en bokning           | 
|  GET    |  /bookings/{id}  |  Hämta en specifik bokning   |
|  GET    |  /bookings/      |  Hämta alla bokningar        |

## Teknologier
- AWS Lambda
- API Gateway
- DynamoDB
- Serverless Framework
